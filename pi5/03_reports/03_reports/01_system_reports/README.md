# System Reports

These folders are for report-style emails: scheduled summaries and run-result mails.

Use them for content such as:

- weekly or monthly summaries
- daily backup result mails
- report mails with PNG or log attachments

Recommended layout:

- one run folder per send
- `email.txt` for the body copy
- copied attachments kept beside it
- `meta.json` when archived by `smtp_archive.py`
